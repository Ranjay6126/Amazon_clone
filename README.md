Amazon Clone 🛒

A responsive Amazon clone built using HTML and CSS, designed to replicate the core layout and visual experience of the Amazon e-commerce platform. This project focuses on front-end structure, styling, and responsive design to closely match the original user interface.

The clone includes a navigation bar, search section, product listings, banners, and footer, all styled with clean and organized CSS. It helped strengthen my understanding of modern layouts, Flexbox, and responsive web design principles.

Features

Responsive homepage layout

Amazon-style navigation bar and footer

Product grid and banner sections

Clean and structured CSS styling

Cross-device compatibility

Tech Stack

HTML5

CSS3

Purpose
This project was created for learning and practice, focusing on UI design, layout structuring, and improving front-end development skills.
